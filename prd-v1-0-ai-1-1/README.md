# 像素马拉松

情侣纪念日计时器，纯静态 Web App，可部署到 GitHub Pages。

## GitHub Pages 部署

1. 在 GitHub 创建一个新仓库，例如 `pixel-marathon`。
2. 上传本目录内所有文件到仓库根目录。
3. 进入仓库 `Settings -> Pages`。
4. 在 `Build and deployment` 中选择 `Deploy from a branch`。
5. Branch 选择 `main`，Folder 选择 `/root`，保存。
6. 等待 1-3 分钟，访问 `https://你的用户名.github.io/pixel-marathon/`。

部署后点击页面右上角分享按钮，即可生成可发给另一台设备的快照链接。
